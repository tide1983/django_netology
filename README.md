Проект для изучения Django с тремя view-функциями:

1. Главная страница (`/`) - список доступных страниц
2. Текущее время (`/current_time/`) - показывает текущее время
3. Рабочая директория (`/workdir/`) - показывает содержимое рабочей директории

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/django_netology.git
cd django_netology

2. Установить зависимости:

pip install -r requirements.txt

3. Запустить сервер:

python manage.py runserver

4. Открыть в браузере: http://127.0.0.1:8000/
